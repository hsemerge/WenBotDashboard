🎁 Giveaway Commands

!giveaway start code
Start a code-user giveaway. Only verified users can enter.
MOD
!giveaway start wager
Start a wager-based giveaway with weighted entries.
MOD
!giveaway start open
Open giveaway — any viewer can enter.
MOD
!giveaway stop
Close entries for the active giveaway.
MOD
!winner
Pick a random winner from current entries.
MOD
!verify [username]
Link your affiliate account to enter giveaways.
ALL

🪙 Points Commands

ALL

ALL

ALL
!top

ALL
!rank
See your current rank on the leaderboard.
ALL
!addpoints @user [n]

MOD
!setpoints @user [n]

🎰 Guess the Balance (!gtb)

!gtb start
Open the GTB session — viewers can now submit guesses.
MOD
!gtb stop
Close the GTB session — no more guesses accepted.
MOD
!gtb [amount]
Submit your guess. e.g. !gtb 4250
ALL
!gtb winner [actual]
Find the closest guess. WenBot announces winner.
MOD
!mygtb

📖 Getting Started

1
Go to Settings and connect your affiliate platform API key
2
In your Kick chat type /mod WenBot to give WenBot mod permissions
3
Viewers type !verify [affiliateUsername] to link their account before entering giveaways
4
Type !giveaway start code in chat to launch your first giveaway
